# Python Network Automation Script 🚀

A robust Python-based automation tool designed to manage network configurations and monitor device status efficiently.

## 🌟 Key Features
- **Automated Connectivity:** Securely connects to network devices using SSH via the Netmiko library.
- **Status Monitoring:** Automatically retrieves and displays interface status and configuration summaries.
- **Error Handling:** Built-in exception handling to manage connection timeouts and authentication errors.

## 🛠️ Built With
- **Python:** Core scripting language.
- **Netmiko:** For multi-vendor network device interaction.
- **Scapy:** For network packet manipulation and scanning.
